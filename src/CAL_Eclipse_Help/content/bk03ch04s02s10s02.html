Examples

data public MyColorEnum =
    Red |
    Green |
    Blue |
    Other
    deriving Enum, IntEnum;

With the above definition of MyColorEnum:

enumToInt Red returns 0

enumToInt Other returns 3

intToEnum 2 returns Blue

intToEnum 500 Error: argument (100) does not correspond to a value of type MyColorEnum
