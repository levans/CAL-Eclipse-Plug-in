Unpacking tuple components using case expressions

See the section called “Matching tuples” for a detailed description of using case expressions to extract components from tuples. Case expressions allow an expression to refer to each component of a tuple by name: 

twoDPointToRec point =
    case point of
    (xVal, yVal) -> {x=xVal, y=yVal};
    ;
