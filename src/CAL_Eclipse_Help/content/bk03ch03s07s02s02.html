Unpacking named fields using case expressions

Another method for handling records in CAL is the case expression (see the section called “Matching records” for a detailed description). The usual situation is to want to extract a specific named field or fields from a record, which may contain other, unspecified fields:

recTo2DPoint rec =
    case rec of
    {_ | x, y} -> (x,y);
    ;
