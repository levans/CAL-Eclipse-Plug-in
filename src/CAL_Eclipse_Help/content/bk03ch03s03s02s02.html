Tuples

A tuple is an ordered, constant-size collection of values of possibly differing types. Tuple types are specified by enclosing a comma-separated list of the types of each component of the tuple in parentheses:

( type_name, type_name [, type_name ...] )

For example:

:t (55.1, 55.2) outputs (Double, Double)

:t (0.9, "foo", 'Z') outputs (Double, String, Char)

:t ('C', 'A', "baz", "qux") outputs (Char, Char, String, String)

:t ('m', ['a', 'c', 'b']) outputs (Char, [Char])

:t (["str1", "str2"], 'c', (2.2, 1.3)) outputs ([String], Char, (Double, Double))
